Delete files older than a certain age in HP-UX

I've only tested this on a HP-UX v11.3 box, but I believe that this should work on any *nix system.

Make a log file of all files that are going to be deleted.

find . -name "example" -mtime +30 -exec ll {} \; > report.log

Command Meaning
find the find command
. Searching in the current directory
-name "example" search by filename
-mtime +30 if the modified time is older than 30 days
-exec ll for each item found print it's LL
{} \; This is the end portion to the find command
> report.log print each LL to this log file

Delete those files

find . -name "example" -mtime +30 -exec rm {} \;
